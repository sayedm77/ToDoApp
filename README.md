# Todo List App 
The todo list app is designed for organizing and prioritizing tasks efficiently.

## Technologies Used

1- Programming Language: Objective-C
2- Framework: UIKit
3- Local Storage: UserDefaults


## Features 
1- Add Tasks.

2- View Tasks.

3- Edit Tasks.

4- Remove Tasks.

5- Mark Task Status.

6- Search Tasks.

7- Filter Tasks.

8- Local Storage.
