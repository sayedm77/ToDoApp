# Todo List App 
The todo list app is designed for organizing and prioritizing tasks efficiently.

## Technologies Used

Programming Language: Objective-C
Framework: UIKit
Local Storage: UserDefaults


## Features 

Add Tasks.
View Tasks.
Edit Tasks.
Remove Tasks.
Mark Task Status.
Search Tasks.
Filter Tasks.
Local Storage.
